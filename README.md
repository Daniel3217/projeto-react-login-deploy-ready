# 🔐 Projeto Google Login + Cadastro (React)

## 📖 Descrição do projeto
Este projeto é uma aplicação web desenvolvida em React que simula um sistema de autenticação com:
- Cadastro de usuários
- Login com email e senha
- Login com conta Google (OAuth)
- Armazenamento de dados no localStorage
- Proteção de rotas (PrivateRoute)
- Interface inspirada no Google

O objetivo do projeto é demonstrar conceitos de autenticação, manipulação de estado e integração com APIs externas.

---

## 🚀 Tecnologias utilizadas
- React JS
- React Router DOM
- JavaScript (ES6+)
- HTML5
- CSS3 (inline styles)
- Google OAuth (@react-oauth/google)
- JWT Decode
- LocalStorage (persistência de dados)
